# 🧊 What's in Your Fridge?

A React Native (Expo) mobile application that helps users discover 
recipes based on ingredients they already have at home.

## 📱 Features
- Search recipes by entering available ingredients
- Fetches real-time recipe data via TheMealDB API
- Displays full recipe details: ingredients + step-by-step instructions
- Clean, minimalist UI with food photography
- User authentication with Firebase

## 🛠 Tech Stack
- React Native (Expo)
- Firebase (Authentication)
- TheMealDB API
- React Navigation

## 🚀 Getting Started
git clone https://github.com/Busrwa/whatIsInMyFridge.git
cd whatIsInMyFridge
npm install
npx expo start
